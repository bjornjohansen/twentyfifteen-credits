Add a credit, e.g. a copyright notice, in the footer of Twenty Fifteen using the theme customizer.

== Description ==
With the default theme from WordPres 4.1, Twentyfifteen, you can add a customized string in the footer. This plugin gives you a input field in the dashboard, so you don’t have to touch any code.

This plugin was created due to a request in the “Norwegian WordPress Admins” Facebook group.

Translations:
* nb_NO

== Installation ==
1. Download and unzip plugin
2. Upload the 'twentyfifteen-credits' folder to the '/wp-content/plugins/' directory,
3. Activate the plugin through the 'Plugins' menu in WordPress.
